This component library is inspired by the [Composition](https://cube.fyi/composition.html) concept in the [CUBE CSS](https://cube.fyi/) methodology that includes layout components whose primary purpose is to take any amount of content or components and put them inside some sort of layout structure.

## How to use?

1. **Make sure to have shadcn/ui configured in your project:**

   The shadcn/ui [documentation](https://ui.shadcn.com/docs/installation) will show you how to do that, if you haven't done it yet.

2. **Add whatever components you want by running the corresponding npm commands in your terminal**
